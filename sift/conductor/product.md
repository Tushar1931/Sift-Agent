# Sift — Product Context

## Tagline
"See yourself the way an attacker does."

## What Sift Does
Sift is a privacy audit tool that reconstructs a user's digital identity
from their LinkedIn and X social media archives. It surfaces what an
attacker could infer about them — including information they never
explicitly posted — and guides them through fixing it.

## Core Value Proposition
Most privacy tools tell you what data you shared. Sift tells you what
an attacker can *infer* from that data — including cross-platform
identity chains that are invisible in isolation.

## Target User (Hackathon Demo)
A fictional persona: Alex Morgan — a Senior Security Engineer based in
Koramangala, Bengaluru. The demo dataset is pre-tuned to reliably
produce this inference.

## Six Product Phases
1. Zero Friction Entry — Landing page with drag-drop upload and Try Demo
2. Instant Gratification — Live file tally and cross-platform indicator
3. Tech Theater — Scrolling AI analysis log, 6 seconds fixed duration
4. The Reveal — Bold identity inference, confidence scores, inference graph
5. Audit & Remediation Workspace — Risk findings + gamified fix checklist
6. Exit Loop — Privacy Receipt PDF, viral share card, session wipe

## Privacy Promise
No data is ever stored or sent to a server. Everything is processed
client-side. Session is wiped on exit.
